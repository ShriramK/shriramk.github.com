angular.attrMarkup

Work in Progress This page is currently being revised. It might be incomplete or contain inaccuracies.

Attribute markup extends the angular compiler in a very similar way as angular.markup except that it allows you to modify the state of the attribute text rather than the content of a node.

Create custom attribute markup like this:

  angular.attrMarkup('newAttrMarkup', function(attrValue, attrName, element){
    //tranformation code
  });

For more information about angular attribute markup, see Understanding Angular Markup in the angular Developer Guide.
