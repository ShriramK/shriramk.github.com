This is a static copy of the SingPath GUI. You can fork this project to create a new version of the SingPath GUI.
This project contains static text files that represent the JSON data that SingPath returns to HTML pages.
by SingaporeClouds
SingPath

This is a copy of the SingPath Angular-based GUI that developers and students can fork and enhance. The goal for this project is to provide everything that would be needed to develop, enhance, and test the SingPath GUI.

Authors

Supported GUI's
kit - the angular-based gui being extended to work statically from Github and Cloud9
alex - the first angular-based gui
shane - static HTML concept pages
api/docs - the documentation for the SingPath api that gui's pull data from.
